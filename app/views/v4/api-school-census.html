School census API v1.0 Download swagger file

Receives the census XML file from the Data Provider, validates and returns a validation report showing the errors and queries. This will over-write any previous census data and supplementary information such as notes.

POST /api/Census

Parameters

  • census_file (File) XML file
  • submit_census (Boolean) : true | false Validates census when 'false'

Headers

Content-Type: application/json
Authorization: Bearer [access token]
X-API-Key: Show school census API key

Example body

{
    "census_file": "st_mary_aut_2018.xml"
    "submit": "false"
}

Example response

"200": {
    "description": "Success"
}

Response codes

  • 200 - Success
  • 400 - Bad Request
  • 401 - Unauthorized
  • 403 - Forbidden
  • 500 - Server Error
