Common basic data set v1.0 Download swagger file

Returns definitions for common data items that schools and local authorities use in certain software systems, for example school management information systems.

POST /api/common-basic-data-set

Parameters

  • CBDSIdentifier (String) Data item name

Example body

{
    "CBDSIdentifier": "N00015"
}

CBDS Identifiers

  • N00015 - Language code
  • N00026 - Exclusion category
  • N00027 - Exclusion reason
  • N00166 - Pupil SEN type code
  • N00177 - Child ethnicity
  • N00209 - SEN provision

Example response

"language: {
    ACL: Acholi,
    ADA: Adangme,
    AFA: Afar-Saho,
    AFK: Afrikaans,
    AKA: Akan/Twi-Fante,
    AKAF: Akan (Fante),
    AKAT: Akan (Twi/Asante),
    ALB: Albanian/Shqip,
    ALU: Alur,
    AMR: Amharic,
    ...
    ZUL: Zulu
}
"200": {
    "description": "Success"
}

Response codes

  • 200 - Success
  • 400 - Bad Request
  • 401 - Unauthorized
  • 403 - Forbidden
  • 500 - Server Error
